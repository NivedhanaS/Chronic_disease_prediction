# Chronic Disease Prediction

## 📌 Overview

This project predicts chronic kidney disease using machine learning.

## 🔧 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 📊 Steps

* Data Cleaning
* Handling Missing Values
* Feature Selection
* Model Training (Logistic, Random Forest, etc.)
* Evaluation (Accuracy, Precision, Recall, F1)

## 🚀 Results

Achieved good performance using multiple ML models.

## ▶️ How to Run

```bash
pip install -r requirements.txt
```

Run notebook:

```bash
jupyter notebook
```

## 📁 Dataset

Kidney Disease Dataset
